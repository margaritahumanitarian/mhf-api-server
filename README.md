## Description

[MHF Api Server](https://github.com/margaritahumanitarian) built to serve the many needs of the foundation.

## Installation

```bash
$ npm install
```

## Setup Database

```md
# .env
- create .env file in the root dir
- use .env.example to populate .env
```
## Running the app

```bash
# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
